# Reconstruction-of-Two-Flavor-Neutrino-Oscillation-Constraints-from-KamLAND-Reactor-Antineutrino-Data
This paper discusses a two-flavor neutrino oscillation analysis of reconstructed reactor antineutrino data from the early KamLAND experiment (2002) using a $(\chi^2)$ based parameter estimation method. A combined rate and spectral shape likelihood was used to determine the oscillation parameters $(\Delta m^2_{12})$ and $(\sin^2 2\theta_{12})$. The best-fit values obtained are $(\Delta m^2_{12}=5.31\times10^{-5},\mathrm{eV}^2)$ and $(\sin^2 2\theta_{12}=0.745)$. Parametric contours at $68.3, 90, 95 and 99 percent confidence levels are presented.
